--------------------------------------------------------------------------------
　DJMAX RESPECT V 曲選択ツール
--------------------------------------------------------------------------------

●どんなもの？
DJMAX RESPECT Vの曲をランダムで表示してくれる簡易ツールです。

●機能
　・ランダムな曲の表示
　　フィルタリングした曲リストの中でランダムで表示してくれます。

　・buttonや難易度でのフィルタリグ
　　1クリックでフィルタリングの設定可能
　　大体の人はこれで満足できるはず！

　・手動更新 or 自動更新
　　1度だけの更新はもちろん、設定した時間での自動更新も可能！
　　推奨は一番短い曲(Funky Chups)の1分32秒。

　・簡易的なOBSとの連携
　　配信者向け。
　　任意のフォルダにランダムで選んだ曲情報をテキストファイルで出力。
　　これをOBSで読み込めば・・・・・


●注意事項
　・素人の作ったものです。
　　システムを壊すことなんてできないとは思いますが、自己責任でお願いします。

　・配布について
　　下部の連絡先に一言いただければ基本許可します。
　
　・自動更新はシステムに影響を及ぼす必要があります。
　　不安を感じたら即PC再起動で！
